# Personal Task Manager Dashboard

A Vanilla JavaScript project built with HTML, CSS, and JavaScript only.

## Features
- Task Display
- Task Filtering
- Search
- Data Persistence
- API Integration
- Statistics

## Tech Stack
- HTML
- CSS
- Vanilla JavaScript
- localStorage
- Fetch API

## Setup Instructions
1. Clone the repository
2. Open the project folder
3. Open `index.html` in browser or use Live Server

## Screenshots
Add ASSETS in the `assets/` folder.
